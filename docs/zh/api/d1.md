# D1 数据库

## GET /cf/:accountId/d1/databases

列出 D1 数据库。

## POST /cf/:accountId/d1/databases

创建数据库。

**请求：**
```json
{ "name": "my-database" }
```

## DELETE /cf/:accountId/d1/databases/:databaseId

删除数据库。

## POST /cf/:accountId/d1/databases/:databaseId/query

执行 SQL 查询。

**请求：**
```json
{ "sql": "SELECT * FROM users WHERE id = ?", "params": ["1"] }
```
