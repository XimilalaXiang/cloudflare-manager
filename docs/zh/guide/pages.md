# Pages

管理 Cloudflare Pages 项目及其部署。

## 查看项目列表

从下拉菜单选择账户，即可查看所有 Pages 项目，包括子域名、自定义域名和生产分支。

## 创建项目

1. 点击 **创建项目**
2. 输入项目名称
3. 可选设置生产分支（默认为 `main`）
4. 点击 **创建**

## 查看项目详情

点击项目卡片查看完整详情，包括项目 ID、子域名、所有关联域名和生产分支。

## 管理部署

每个项目卡片上有 **部署** 按钮，点击可查看所有部署信息：

- 部署 ID 和短 ID
- 环境（production / preview）
- 部署 URL
- 最新阶段状态
- 创建日期

可以在此视图中删除单个部署。

## 删除项目

点击项目卡片上的 **删除** 按钮并确认，将移除项目及其所有部署。
