# 邮件路由

Cloudflare 邮件路由允许你为域名创建自定义邮箱地址，并将收到的邮件转发到任意目标邮箱 —— 完全免费。

## 概述

邮件路由是**域名级别**的功能。选择一个域名后可管理：

- **设置** — 启用或禁用该域名的邮件路由
- **路由规则** — 定义哪些地址转发到哪些目标
- **全匹配规则** — 处理不匹配任何特定规则的邮件
- **目标地址** — 管理已验证的目标邮箱（账户级别）

## 使用邮件路由

### 1. 选择账户和域名

选择一个 Cloudflare 账户，然后选择域名。邮件路由的设置和规则是按域名配置的。

### 2. 目标地址

创建路由规则前，需要在账户级别添加目标地址。每个地址会收到 Cloudflare 发送的验证邮件。

### 3. 启用邮件路由

为选定的域名开启或关闭邮件路由。

### 4. 创建规则

每条规则包含：
- **匹配条件** — 匹配收件邮件的条件（如收件人地址）
- **动作** — 对匹配邮件执行的操作（如转发到目标地址）
- **优先级** — 规则按优先级顺序执行

### 5. 全匹配规则

全匹配规则处理不匹配任何特定规则的邮件，可在面板中查看其当前配置。
