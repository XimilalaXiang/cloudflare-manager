# Pages

Manage Cloudflare Pages projects and their deployments.

## Listing Projects

Select an account from the dropdown to see all Pages projects with their subdomain, custom domains, and production branch.

## Creating a Project

1. Click **CREATE PROJECT**
2. Enter the project name
3. Optionally set the production branch (defaults to `main`)
4. Click **CREATE**

## Viewing Project Details

Click on a project card to view its full details including ID, subdomain, all associated domains, and production branch.

## Managing Deployments

Each project card shows a **DEPLOYMENTS** button. Click it to see all deployments with:

- Deployment ID and short ID
- Environment (production / preview)
- Deployment URL
- Latest stage status
- Creation date

You can delete individual deployments from this view.

## Deleting a Project

Click **DELETE** on any project card and confirm the deletion. This will remove the project and all its deployments.
